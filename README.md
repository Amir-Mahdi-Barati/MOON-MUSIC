# 📖 Overview

**MOON MUSIC** is a modern desktop music player designed with performance, simplicity, and a futuristic user experience in mind.

The goal of this project is to create a lightweight yet feature-rich music application that combines elegant UI design with powerful playback capabilities.

Whether you're listening casually or managing a large music library, MOON MUSIC is built to deliver a smooth experience.

---

# ✨ Features

- 🎵 High-quality audio playback
- ⚡ Fast and lightweight
- 🎨 Modern user interface
- 🌙 Dark Mode
- 📂 Playlist Management
- 🔍 Smart Search
- ❤️ Favorites
- 🎧 Local Music Library
- 📁 Drag & Drop Support
- 🔊 Volume & Playback Controls
- ⏯ Shuffle / Repeat
- 🚀 Optimized Performance
- 🛠 Open Source

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/Amir-Mahdi-Barati/MOON-MUSIC.git
```

Go to project

```bash
cd MOON-MUSIC
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run

```bash
python main.py
```
